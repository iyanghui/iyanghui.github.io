angular2入门之动画指南

发表于 2017-06-13 | 分类于 angular2 | 阅读次数

前言

  写这篇文章的时候所使用的是angular2.X,而ng2.x动画相关的api是并入@angular/core这个核心模块的，但是在angular4之后开始独立，引入的方式也有所改。

关于ng2动画的理解

  • 按照官网上来说，就是在两个状态之间进行转。angular通过在一定时间内将组件内的css样式过渡切换成另一种样式来实现动。是由状态和状态之间的转场效果所定义。动画被定义在@Component元数据。
  • 需要定义一个动画触发器(triggerName)，在模板中使用[@triggerName]语法来把它附加到一个或多个元素上.triggerName设置成表达式，通过不同的状态，来定义动画状态
  • state中具体定义的是每个状态的最终样。一旦元素转场到这个状态，那么样式就会留在这个状态，并且会一直保持。从这个意义上讲，这里其实并不只是定义动画，而是在定义该元素在不同状态时应该具有的样。 <!-- more -->

使用ng2动画

  1. 例子讲解<br/>   本例包含不同状态进场和立场(基于关键帧(Keyframes)的多阶段动画，类似于ccss3中的动画，使用关键帧来为进场和离场动画添加一些“反弹效果”）<br/>   通过定义动画的关键帧，可以把两组样式之间的简单转场，升级成一种更复杂的动画，它会在转场期间经历一个或多个中间样式。每个关键帧都可以被指定一个偏移量，用来定义该关键帧将被用在动画期间的哪个时间点。偏移量是一个介于0(表示动画起点)和1(表示动画终点)之间的数组。

总结

  1. trigger(触发) 首先动画是包在触发器里。所以我们要先定义一个trigger<br/>
  • 说明: trigger()方法接受两个参数：<br/>

params1:String 动画标识符<br/> params2:Array 多个状态、转场的数。

  trigger('animateName',[state(),state()...,transition(),transition...]);

  • 使用
  1. state（状态）<br/> state比较好理解，就是某一状态下元素有不同的样。例如左边飞入动画，它有两个状态，一个是在屏幕左边看不见的位置，另一个在屏幕中显示的位置，这就是两个状。从第一个状态到第二个状态的转场也就形成了动。
  • 说明: state()方法接收两个参数：<br/>

param1:String 状态名称<br/> param2:Object style方法   state('stateName',style({}))

  • 使用
  1. transition（转场）<br/> 转场也就是由一个状态过度到另外一个状态的过。
  • 说明: transition()方法接受两个参数：<br/>

param1:String 状态1 => 状态2<br/> param2:Object animate方法

  • 使用

<span style="color: red">注意</span>

  1. *（通配符）状态<br/> *状态匹配任何动画状。当定义那些不需要管当前处于什么状态的样式及转场时，例如新增item。
  2. void状态<br/> void状态表示元素没有被附加到视。这种情况可能是由于它尚未被添加进来或者已经被移除。
